# weather-app
Key Features:

Uses the free Open-Meteo and BigDataCloud APIs to pull real-time weather and location data (no API keys required!).

A custom-built CSS compass needle that mathematically calculates and rotates to the live wind direction.

A fully functional search bar that catches errors and resets the screen if a city isn't found.

Completely responsive and locked to the center of the screen for a native-app feel.
